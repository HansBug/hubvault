Installation
===================

hubvault is currently hosted on PyPI. It requires Python >= 3.7 and is tested on CPython 3.7 through 3.14.

You can simply install hubvault from PyPI with the following command:

.. code:: shell

    pip install hubvault

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/hansbug/hubvault@main
