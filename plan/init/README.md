# init 计划说明

本目录保存的是 `hubvault` 在“从无到有”阶段形成的初始方案文档。

这一批文档的定位是：

- 给出项目的第一版技术边界
- 固定最核心的协议与一致性假设
- 为后续实现阶段提供统一基线

这些文档不是任务清单，而是初始化阶段的设计蓝图。后续如果进入实现、重构、性能优化或专项方案讨论，建议在 `plan/` 下另建新子目录，避免与本目录混用。
